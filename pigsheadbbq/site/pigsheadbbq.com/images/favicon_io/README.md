# Favicon assets (manual)

Binary favicon files are intentionally not committed to this repository.

Before deploy, ensure these files exist in this folder:

- `apple-touch-icon.png` (180x180)
- `favicon-32x32.png` (32x32)
- `favicon-16x16.png` (16x16)
- `favicon.ico`
- `android-chrome-192x192.png` (192x192)
- `android-chrome-512x512.png` (512x512)

The HTML templates and manifest already reference these paths.
